# Лабораторная работа №7: Многопоточное программирование

## Описание
В работе изучены принципы многопоточного программирования в Java: создание и управление потоками с помощью `ExecutorService`, получение результатов через `Future`, синхронизация с `CountDownLatch` и защита общих ресурсов.

---

## Задание 1: Сумма элементов массива

### Реализация
Программа `ArraySumThreads` вычисляет сумму элементов массива из 100 чисел (от 1 до 100) с использованием `ExecutorService`.

- Массив разделён на 4 равные части.
- Каждая часть обрабатывается отдельным потоком (задача реализована через `Callable<Integer>`).
- Результаты возвращаются в виде объектов `Future`.
- Главный поток суммирует результаты всех потоков.

---

## Задание 2: Поиск наибольшего элемента в матрице

### Реализация
Программа `MatrixMaxThreads` находит максимальный элемент в матрице с использованием многопоточности.

- Матрица делится по строкам между 3 потоками.
- Каждый поток находит максимум в своей группе строк.
- Главный поток из полученных максимумов определяет глобальный максимум.

---

## Задание 3: Моделирование работы склада

### Реализация
Программа `Warehouse` моделирует работу трёх грузчиков, которые переносят товары до достижения лимита 150 кг.

- Использован `CountDownLatch` с начальным значением 3 для ожидания завершения всех грузчиков.
- Доступ к общему индексу товаров синхронизирован через `synchronized`.
- Каждый грузчик последовательно забирает товары, пока не превысит лимит или не закончатся товары.
- После завершения грузчик вызывает `latch.countDown()`.
