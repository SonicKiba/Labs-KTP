##  Содержание курса

| Лабораторная | Тема | Что реализовано |
|-------------|------|----------------|
| **lab1** | Алгоритмические основы | Генерация простых чисел, проверка чисел-палиндромов |
| **lab2** | ООП (наследование, полиморфизм) | Модель иерархии сотрудников: `Employee` → `Manager`, `Programmer`, `Administrator` |
| **lab3** | Хеш-таблицы | Система учёта заказов интернет-магазина |
| **lab4** | Исключения | Обработка ошибок в трёх сценариях: расчёт среднего, копирование файлов, защита от деления на 0 |
| **lab5** | Регулярные выражения | Утилиты для валидации: поиск чисел, проверка пароля, IP-адреса, слов по первой букве |
| **lab6** | Коллекции | Топ-10 частотных слов, реализация стека, трекер продаж |
| **lab7** | Многопоточность | Параллельное суммирование массива, разделение на чёт/нечет, симуляция склада с барьером |
| **lab8** | Рефлексия + аннотации | Многопоточная обработка данных с кастомными аннотациями |
