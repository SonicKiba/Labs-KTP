# Лабораторная работа №5: Регулярные выражения

## Описание
В работе изучены основы работы с регулярными выражениями в Java: классы `Pattern` и `Matcher`, применение регулярных выражений для поиска, проверки и обработки текстовой информации.

---

## Задание 1: Поиск всех чисел в тексте

### Реализация
Программа `NumberFinder` ищет все числа (целые и десятичные) в заданном тексте с использованием регулярного выражения `\\d+\\.?\\d*`. Найденные числа выводятся на экран.

---

## Задание 2: Проверка корректности пароля

### Реализация
Программа `PasswordValidator` проверяет пароль на соответствие требованиям:
- длина от 8 до 16 символов,
- только латинские буквы и цифры,
- минимум одна заглавная буква,
- минимум одна цифра.

Регулярное выражение: `^(?=.*[A-Z])(?=.*\\d)[A-Za-z0-9]{8,16}$`

---

## Задание 3: Поиск сочетаний «строчная + заглавная» буква

### Реализация
Программа `CaseFinder` находит в тексте все случаи, когда сразу после строчной буквы идёт заглавная, и выделяет их знаками `!` с двух сторон. Используется регулярное выражение `([a-z])([A-Z])`.

---

## Задание 4: Проверка корректности IP-адреса

### Реализация
Программа `IPValidator` проверяет, является ли введённая строка корректным IPv4-адресом (четыре числа от 0 до 255, разделённые точками). Регулярное выражение учитывает все допустимые диапазоны для каждого октета.

---

## Задание 5: Поиск слов по первой букве

### Реализация
Программа `WordFinder` запрашивает у пользователя букву и находит в тексте все слова, начинающиеся с этой буквы (регистр не учитывается). Регулярное выражение: `\\b` + буква + `\\w*` с флагом `Pattern.CASE_INSENSITIVE`.
