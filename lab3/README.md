## Описание

## Задание 1: Собственная HashTable

### Реализация
Создан обобщённый класс `HashTable<K, V>`, который хранит данные в массиве связных списков `LinkedList<Entry<K, V>>[]`.

**Внутренний класс Entry<K, V>** — хранит пару «ключ-значение».

**Методы:**
- `put(K key, V value)` — добавляет пару; если ключ уже существует, обновляет значение.
- `get(K key)` — возвращает значение по ключу или `null`.
- `remove(K key)` — удаляет пару по ключу, возвращает `true`/`false`.
- `size()` — возвращает количество элементов.
- `isEmpty()` — проверяет, пуста ли таблица.
- `hash(K key)` — вычисляет индекс массива: `Math.abs(key.hashCode() % CAPACITY)` (ёмкость по умолчанию — 10).

**Разрешение коллизий:** метод цепочек — элементы с одинаковым хэшем попадают в один список.

## Задание 2: Учёт сотрудников компании (HashMap)

### Класс Employee
Поля: `name` (имя), `position` (должность), `salary` (зарплата).  
Реализованы:
- Конструктор с параметрами.
- Геттеры для всех полей.
- Сеттеры для `position` и `salary`.
- `toString()` для вывода информации.

### Класс CompanyEmployees
Внутри используется `HashMap<Integer, Employee>`, где ключ — ID сотрудника.

**Методы:**
- `addEmployee(int id, Employee employee)` — добавляет сотрудника.
- `getEmployee(int id)` — возвращает сотрудника по ID или `null`.
- `removeEmployee(int id)` — удаляет сотрудника, выводит сообщение.
- `printAllEmployees()` — выводит всех сотрудников и их количество.
