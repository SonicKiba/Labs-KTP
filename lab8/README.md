# Лабораторная работа №8: Аннотации, рефлексия и многопоточная обработка данных

## Описание
В работе изучены механизм аннотаций, рефлексия для динамического вызова методов, Stream API и многопоточная обработка данных с использованием `ExecutorService`.

---

## Реализация

### Аннотация @DataProcessor
Создана пользовательская аннотация с временем жизни `RUNTIME` и параметром `description()`, позволяющая помечать методы-обработчики данных.

### Класс DataManager
Основной класс, управляющий обработкой данных:
- Регистрация объектов-обработчиков.
- Загрузка данных из файла с помощью `java.nio.file.Files`.
- Многопоточная обработка: через рефлексию находятся методы, помеченные `@DataProcessor`, и каждый запускается в отдельном потоке с помощью `ExecutorService`.
- Сохранение результата в выходной файл.

### Классы-обработчики
Реализованы три класса с аннотированными методами:
- `UpperCaseProcessor` — преобразует текст в верхний регистр.
- `ReverseProcessor` — переворачивает текст задом наперёд.
- `FilterProcessor` — фильтрует строки, оставляя только те, длина которых больше 10 символов (с использованием Stream API).

### Главный класс Main
Демонстрирует работу приложения:
- Регистрирует все обработчики.
- Загружает данные из файла `input.txt`.
- Запускает многопоточную обработку.
- Сохраняет результат в `output.txt`.
