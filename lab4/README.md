# Лабораторная работа №4: Обработка исключений

## Описание
В работе изучен механизм обработки исключений в Java: блоки try-catch-finally, создание собственных классов исключений, логирование ошибок, а также обработка исключений при работе с массивами и файлами.

---

## Задание 1: Среднее арифметическое массива

### Реализация
Программа `ArrayAverage` вычисляет среднее арифметическое элементов массива.

- Использован блок `try-catch-finally`.
- Обрабатывается `ArrayIndexOutOfBoundsException` (выход за границы массива).
- Предусмотрен общий перехват `Exception`.

---

## Задание 2: Копирование файла

### Реализация
Программа `FileCopy` копирует содержимое файла `input.txt` в `output.txt`.

- Использован `try-with-resources` для автоматического закрытия потоков.
- Обрабатываются `FileNotFoundException` (файл не найден) и `IOException` (ошибки ввода-вывода).

---

## Задание 3: Математические операции с пользовательским исключением (Вариант 8)

### Реализация

**CustomUnsupportedOperationException** — собственный класс исключения, наследуемый от `Exception`. Выбрасывается при попытке выполнить неподдерживаемую операцию.

**ExceptionLogger** — класс для логирования исключений в файл `exceptions.log`. Записывает:
- дату и время,
- имя класса,
- тип исключения,
- сообщение,
- дополнительную информацию,
- стектрейс.

**MathOperations** — программа, выполняющая операции `+`, `-`, `*`, `/`. При попытке использовать неподдерживаемую операцию (например, `%`) выбрасывается `CustomUnsupportedOperationException`. Все исключения логируются через `ExceptionLogger`.
