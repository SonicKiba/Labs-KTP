# Лабораторная работа №6: Коллекции и дженерики

## Описание
В работе изучены основные коллекции Java: `Map`, `List`, `HashMap`, `ArrayList`, а также обобщённое программирование (дженерики). Реализованы собственная структура данных `Stack<T>` и программы для анализа текста и учёта продаж.

---

## Задание 1: Топ-10 самых частых слов в файле

### Реализация
Программа `TopWords` считывает текстовый файл `text.txt` и подсчитывает частоту каждого слова с помощью `HashMap<String, Integer>`. После подсчёта данные сортируются по убыванию частоты, и выводится топ-10 самых часто встречающихся слов.

---

## Задание 2: Обобщённый класс Stack<T>

### Реализация
Создан обобщённый класс `Stack<T>`, реализующий стек на основе динамического массива.

**Методы:**
- `push(T element)` — добавляет элемент на вершину стека.
- `pop()` — удаляет и возвращает верхний элемент; выбрасывает исключение, если стек пуст.
- `peek()` — возвращает верхний элемент без удаления.
- `isEmpty()` — проверяет, пуст ли стек.
- `size()` — возвращает количество элементов.

При переполнении массива размер автоматически увеличивается.

## Задание 3: Учёт продаж в магазине (Вариант 8)

### Реализация
Программа `SalesAccounting` использует `ConcurrentHashMap<String, Integer>` для хранения количества проданных единиц по каждому товару.

**Функции:**
- Добавление продажи (название товара и количество).
- Просмотр списка всех товаров с количеством.
- Подсчёт общего количества проданных товаров.
- Определение самого популярного товара (с максимальным количеством продаж).

Программа реализована в виде консольного меню с обработкой ввода.
